# ARVR Group - Nicolas SUNG, Maxence RAULT, Kilian Delcenserie

## Touches (manette)

- `Y` : ouvrir l'inventaire.
- Sélection dans l'inventaire : viser avec la gâchette droite + utiliser la gâchette gauche.
- Fermer l'inventaire puis `gâchette droite` : placer l'objet sélectionné.
- `A` : sauter et ouvrir la porte (FAHHHHHH).
- `Joystick` : se déplacer.
- `X` : reset de la scène.
- `Gâchette intérieure droite` : grab/attraper un objet.
- `Gâchette droite` sur le bouton à côté de la porte : éteindre la lumière.

## Features

- Système d'inventaire avec sélection et placement d'objets.
- Interactions environnementales (porte, bouton, lumière).
- Respawn automatique en cas de chute dans le vide.
- Déplacements et actions de base en VR à la manette.
- Post-traitement visuel type Fallout post-nucléaire.
- Ambiance sonore intégrée (sons du jeu).

## Notes

- Ce projet est orienté expérience AR/VR avec interactions immersives.
